# 🌸 Iris Flower Classification using Machine Learning

A Machine Learning web application that predicts the species of an Iris flower based on its physical measurements. The model is trained using the famous Iris dataset and deployed with Streamlit.

---

## 🚀 Live Demo

🔗 https://YOUR-STREAMLIT-LINK.streamlit.app/

---

## 📌 Features

- Predicts Iris flower species
- Interactive sliders for flower measurements
- Displays prediction confidence
- Modern dark-themed user interface
- Fast real-time predictions

---

## 🌼 Predicted Classes

- 🌼 Iris Setosa
- 🌺 Iris Versicolor
- 🌸 Iris Virginica

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Streamlit
- NumPy
- Pandas

---

## 🤖 Machine Learning Model

- Algorithm: K-Nearest Neighbors (KNN)
- K Value: 4
- Dataset: Iris Dataset
- Framework: Scikit-learn

---

## 📂 Dataset

The project uses the built-in **Iris Dataset** available in Scikit-learn.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Iris-Flower-Classification-ML.git
```

Navigate to the project

```bash
cd Iris-Flower-Classification-ML
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
